JavaScript理论整理

发表于 2017-12-11

整合《高程》和视频所学习到一些方法

基本概念

ECMAScript

一套标准,所谓的规范。不在ES规定之内的东西就会报错。

DOM

数学意义上的“树”

BOM

BOM（Browser Object Document）即浏览器对象模型。

DOM管元素和节点，而BOM管类硬件的东西，如定时器、offset系列方法、scroll系列方法、client系列方法等。

一些方法（实时更新）

typeof 判断数据类型

charAt() 可以找一个字符串里的一个“字”，字符串里的每个字都有一个下标。

Number() 转换为数字 把数据类型转换为number

parseInt() 从第一位开始到非数字为止 的数字。 number的升级版。

parFloat() 从第一位到第二位小数点之前的数字。 可转换成浮点小数。parseInt的升级版。

NaN—not anumber 用来判断是不是数字

isNaN 用来判断是不是数字（奇怪的数字类型）

return
1) 函数名+括号 ： fn1() ==> return 后面的值；
2）return默认为undefined 无添加任何值时；
3）return后面所有的代码都不再执行。

var str = ‘妙味课堂-www.miaov.com’;
str.charAt(1); // ‘味’ 字符串下标寻找
str.charCodeAt(2); // 21619 Unicode编码
string.fromCharCode(22937,21619) // ‘妙味’

str.indexOf(‘m’,3); // 9 从第三位开始查找
str.lastIndexOf(‘o’); // 16 从最后一位开始查找

‘1000’ < ‘2’ // true 字符串比较规则 比较第一位数再过第二位比较
‘1000’ > 2 // true 字符串和数字比较 转换为number比较

str.substring(0,4); //‘妙味课堂’ 从哪一个下标开始到哪一个下标
str.slice(-3); // ‘com’ 反方向查找

str.toUpperCase();
str.tolowerCase(); 英文大小写切换

str.split(‘.’,2) // 根据前一位进行切割，后一位决定返回几段

arr.join(‘.’); 数组转字符串
arr.push 往数组内添加同时返回该值的下标
arr.unshift 往数组第一位添加同时返回数组长度 ！ IE6 IE7不支持unshift
arr.pop 删除数组最后一个同时返回删除的值
arr.shift 删除数组第一位并返回删除的值

组合使用 unshift（arr.pop（））； 删除最后一位并把它添加到第一位

splice 具有删除、替换、添加功能

sort 排序 ！是以字符串的大小来排序的。
让它以number来排序 可以加一个比较大小的函数
